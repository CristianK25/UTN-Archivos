🎯 Demo Interactiva de Posiciones CSS

Desplázate por la página para ver cómo interactúan las diferentes posiciones

1. POSITION: STATIC (Por defecto)
STATIC 1
STATIC 2

Los elementos siguen el flujo normal del documento.

STATIC: Es la posición por defecto. Los elementos se colocan en el orden que aparecen en el HTML. No se ve afectado por top, bottom, left, right.
2. POSITION: RELATIVE
POSICIÓN ORIGINAL
RELATIVE (top: 30px, left: 50px)

La caja verde se movió desde su posición original, pero conserva su espacio.

RELATIVE: Se mueve respecto a su posición original, pero mantiene su espacio en el flujo del documento. Otros elementos no lo ocupan.
3. POSITION: ABSOLUTE
NORMAL FLOW
ABSOLUTE (top: 20px, right: 20px)

La caja naranja se posiciona respecto al contenedor padre (position: relative).

ABSOLUTE: Se posiciona respecto al ancestro más cercano que tenga position diferente a static. Se retira del flujo del documento.
4. POSITION: FIXED
FIXED (top: 20px, right: 20px)

La caja morada permanece fija en la pantalla mientras haces scroll. ¡Mira la esquina superior derecha! 👆

FIXED: Se posiciona respecto a la ventana del navegador. Permanece fijo al hacer scroll. Perfecto para menús o botones que siempre deben estar visibles.
5. POSITION: STICKY
STICKY (top: 20px) - ¡Desplázate hacia abajo!

🔽 SIGUE DESPLAZÁNDOTE 🔽

La caja amarilla se "pegará" cuando llegue a top: 20px de la ventana

¿Ya se pegó? ¡Sigue bajando para ver más ejemplos!

STICKY: Es una combinación de relative y fixed. Se comporta como relative hasta que alcanza una posición específica, entonces se "pega" como fixed.
